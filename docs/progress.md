# 개발 진행 로그

### 결정 사항
- **자막 수집 도구**: yt-dlp (YouTube API 직접 접근 제한으로 인해 변경)
- **대상 유튜버**: 하정훈의 삐뽀삐뽀 119 소아과
- **개발 접근**: End-to-End 프로토타입 먼저 (1개 영상 → 전체 흐름 검증 → 데이터 확장)
- **검색 방식**: 키워드 기반 + Claude 관련성 판단 (임베딩 API 미사용)
- **저장 방식**: 인메모리/파일 기반 JSON
