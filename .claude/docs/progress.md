## 메모

### RAG 시스템 구조
```
[사용자 질문]
      ↓
[벡터 DB에서 관련 스크립트 검색]  ← 직접 구축
      ↓
[검색된 스크립트 + 질문을 LLM에 전달]  ← Claude API
      ↓
[LLM이 스크립트 기반으로 답변 생성]
```

### 핵심 원칙
- 검색된 스크립트에 근거가 없으면 답변 보류
- 단일 전문가(유튜버)의 관점만 반영
- 일반 상식 혼합 금지
