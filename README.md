# Good Morning My Son

육아 전문가의 콘텐츠만을 지식 소스로 사용하는 **출처 기반 답변 엔진**

## 핵심 철학

> **AI가 똑똑해 보이는 것보다, 정직하게 말 안 하는 것이 더 중요하다**

- 근거가 없으면 추측하지 않고 답변 보류
- 단일 전문가의 관점으로 일관된 답변 제공
- 일반 상식 혼합 금지

## 시스템 구조

```
[사용자 질문]
      ↓
[질문 정제 / 의도 파악]
      ↓
[전문가 스크립트 검색]
      ↓
[관련성 판단]
   ↙        ↘
[충분]     [부족]
   ↓          ↓
[답변 생성]  [답변 보류]
```

## 기술 스택

- **Language**: Kotlin
- **Framework**: Spring Boot
- **자막 수집**: yt-dlp
- **LLM**: Claude API

## 프로젝트 구조

```
src/main/kotlin/com/goodmorning/
├── controller/   # REST API
├── subtitle/     # 자막 수집/파싱
├── embedding/    # 임베딩 생성
├── vectordb/     # 벡터 저장소
└── rag/          # RAG 파이프라인
```

## MVP 범위 (v1)

**포함**
- 유튜버 1명 (하정훈 선생님)
- 영상 50~100개 스크립트
- 답변 생성 또는 보류 판정

**제외**
- 다중 전문가
- 일반 상식 보완
- 감정 공감형 응답

## 확장 가능성

출처 기반 답변 엔진으로서 다음 영역으로 확장 가능:
- 법률 상담
- 의료 정보
- 교육 콘텐츠
- 코칭/멘토링
